Step 2: Add the ajax view to the cal plugin
============================================================

.. include:: ../../Includes.txt

In the calendar base content plugin you have to allow the “Ajax (requires:calajax)” view and make it the default view by moving it to the top of the allowed views. For fe-editing, you also have the according views to be allowed!
